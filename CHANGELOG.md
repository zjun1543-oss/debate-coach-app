### ✨ 核心功能

#### 1. 双训练模式
- **教学模式 (Tutorial Mode)**: 苏格拉底式引导，AI 作为教练
  - 可选训练重点: Clarity (清晰表达) / Rebuttal (有效反驳) / Arguments (构建论点)
  - 引导式提问，不直接完成学生论点

- **对练模式 (Sparring Mode)**: 纯辩论对手，AI 作为对手
  - 期间不进行任何教学
  - 点击 "End Sparring" 生成复盘报告

#### 2. 语音交互 🆕
- **Deepgram 流式语音识别**: 实时转录，无重复问题
- **情绪分析**: 检测说话时的情绪 (happy, sad, angry, fear, surprise, neutral)
- **浏览器内置备用**: 无 API Key 时的回退方案
- **语音合成 (TTS)**: 可选择多种自然语音
- **打断功能**: 按下语音键自动打断 AI 说话
- **一键发送**: 第一次点击开始录音，第二次点击停止并发送

#### 3. 练习历史记录
- **历史记录面板**: 右侧滑入式面板查看所有练习记录
- **记录内容**: 日期、模式、话题、立场、回合数、发言内容、复盘报告
- **自动保存**: 开始新会话时自动保存当前会话
- **详细查看**: 点击展开查看完整的复盘报告
- **删除功能**: 可删除单条记录
- **存储上限**: 最多保存 50 条记录
- **徽章提示**: Header 显示练习次数

#### 4. API 集成
- **Deepgram API**: 语音识别 + 情绪分析 (免费 200h/月)
- **DeepSeek API**: AI 教练对话生成
- **本地存储**: API Keys 安全存储在浏览器

#### 5. 话题管理
- 内置经典 THBT 辩题库
- 自定义话题输入
- 本地缓存保存话题

#### 6. 复盘报告系统 🆕
- **Aura Check (气场仪表盘)**: 流畅度 / 脱水率 / 语速 / **情绪**
- **情绪时间线**: 可视化整个练习的情绪变化
- **Logic Check (逻辑体检)**: 证据链诊断
- **Golden Rewrite (金句重写)**: 原始表达 vs 升级版本

### 🎯 教学方法论

基于 **Bo Seo 辩论方法论**:
1. **Aura First**: 气场优先，关注表达自信度
2. **Logic Layer**: 确保论点包含 Warrant (为什么真实) + Impact (为什么重要)
3. **Real Talk**: 温和指出冗余词 (um, uh, like)
